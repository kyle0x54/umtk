**umtk** is an open source python library for medical image processing.

## Installation

*Install umtk*

```shell
$ git clone git@github.com:kyle0x54/umtk.git
$ cd umtk
$ pip install -e .
```

## Try your first umtk program

```python
>>> import umtk
>>> umtk.isdicom('001.dcm')
True
```
